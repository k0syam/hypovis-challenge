# hypovis-challenge
仮説立て・オンライン上でのデータあつめ・データの可視化・振り返りを制限時間内にやるチャレンジ

# Background
世界中のプロジェクトによって，さまざまなデータと可視化のツールにアクセスできるようになった．  
しかしデータの可視化は仮説と切り離すことが難しい．  
データ可視化にいたるプロセスを上記のツールでおこなうことで，仮説検証の経験値を積みたい．
